As an alternative to the StateFace code, you can use CSS classes that correspond to a state's two-letter USPS abbreviation, either to replace the text of an element with a state's shape, or to prepend to it.

Replacement

<span class="state state-replace state-{abbrev}">{state}</span>

Example:

<span class="state state-replace state-md">Maryland</span> → Maryland

Prepend

<span class="state state-{abbrev}">{state}</span>

Example:

<span class="state state-md">Maryland</span> → Maryland
